## 🎯 核心功能说明

### 录音 + 语音识别同步
```
用户点击"开始录音"
    ↓
同时启动：
  1. AudioRecordingService 录制音频
  2. VoiceRecognitionService 语音识别
    ↓
用户说话
    ↓
  - 录音：保存音频到文件
  - 识别：尝试转换为文本和命令
    ↓
用户点击"停止录音"
    ↓
  - 录音文件保存到：
    /storage/emulated/0/Android/data/com.example.myapplication/files/VoiceRecordings/
  - 文件名：voice_yyyyMMdd_HHmmss.m4a
    ↓
用户可以导出录音核对
```

### 语言自动降级流程
```
启动应用
    ↓
尝试中文识别（zh-CN）
    ↓
检测到 ERROR_CLIENT（语言包不可用）
    ↓
自动切换到英文（en-US）
    ↓
使用英文继续识别
```

## 📊 文件格式

### 录音文件
- **格式**：MPEG-4 Audio (.m4a)
- **编码**：AAC
- **采样率**：16,000 Hz
- **比特率**：128 kbps
- **文件大小**：约 960 KB/分钟

### 文件命名
```
voice_20260201_193045.m4a
       ↑       ↑
      日期    时间
```

## 🚀 使用方法

### 在模拟器上
1. 点击"🎙 开始录音"
2. 使用快捷按钮测试功能（拍照、查询电池等）
3. 点击"⏹ 停止录音"
4. 导出录音文件：
   ```bash
   adb pull /storage/emulated/0/Android/data/com.example.myapplication/files/VoiceRecordings/ ~/Desktop/
   ```

### 在真实设备上
1. 安装APK
2. 授予相机和麦克风权限
3. 点击"🎙 开始录音"
4. 说出中文命令：
   - "拍照"
   - "暂停录像"
   - "查询电池"
   - 等等
5. 点击"⏹ 停止录音"
6. 查看识别结果和录音文件

## 🎓 代码示例

### 开始录音
```kotlin
// 在ViewModel中
fun startAudioRecording() {
    val filePath = audioRecordingService.startRecording()
    if (filePath != null) {
        _uiState.value = _uiState.value.copy(
            isAudioRecording = true,
            audioRecordingFile = filePath
        )
        ttsService.speak("开始录音")
    }
}
```

### 停止录音
```kotlin
fun stopAudioRecording() {
    val filePath = audioRecordingService.stopRecording()
    if (filePath != null) {
        _uiState.value = _uiState.value.copy(
            isAudioRecording = false
        )
        ttsService.speak("录音已保存")
        updateAudioRecordingCount()
    }
}
```

### 语言自动降级
```kotlin
override fun onError(error: Int) {
    // 检测语言包错误
    if (error == SpeechRecognizer.ERROR_CLIENT &&
        currentLanguage == "zh-CN" &&
        !fallbackToEnglish) {

        Log.w(TAG, "检测到中文语言包不可用，自动切换到英文")
        fallbackToEnglish = true
        currentLanguage = "en-US"
    }
    // ... 其他处理
}
```
