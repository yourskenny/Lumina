## 已知问题

### ⚠️ 问题1: "录像失败: null"
**现象**：
```
E CameraViewModel: 录像失败: null
```

**分析**：
- 位置：`VideoRecordEvent.Finalize` 事件处理
- 触发条件：`event.hasError() == true`
- 错误原因为null：`event.cause?.message == null`

**影响**：
- 不影响拍照功能
- 不影响AI分析
- 可能是录像停止时的正常事件被误判为错误

**建议修复**（可选）：
```kotlin
if (event.hasError()) {
    val cause = event.cause
    if (cause != null) {
        // 真正的错误
        val message = "录像失败: ${cause.message}"
        Log.e(TAG, message)
    } else {
        // 正常停止，不是错误
        Log.d(TAG, "录像正常停止")
    }
}
```
