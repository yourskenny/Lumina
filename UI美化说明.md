## 🎨 视觉设计

### Material Design 3 风格

- **色彩系统**：
  - Primary Container：状态卡片背景
  - Secondary Container：浮动按钮
  - Surface：抽屉背景

- **圆角和阴影**：
  - 浮动按钮：6dp elevation
  - 状态卡片：圆角卡片设计
  - 抽屉：8dp elevation

### 动画效果

- **滑入动画**：右侧滑入
  ```kotlin
  slideInHorizontally(initialOffsetX = { it })
  ```

- **滑出动画**：右侧滑出
  ```kotlin
  slideOutHorizontally(targetOffsetX = { it })
  ```

- **背景渐变**：半透明黑色背景

### 响应式布局

- **抽屉宽度**：320dp（固定）
- **按钮布局**：2列网格
- **可滚动**：内容超出时可上下滚动
- **间距统一**：8dp 按钮间距，16dp 内边距

## 🚀 使用方法

### 在Windows设备上测试

```powershell
# 1. 切换到ui-improvement分支
git checkout ui-improvement

# 2. 拉取最新代码
git pull origin ui-improvement

# 3. 编译安装
./gradlew clean assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk

# 4. 启动应用
adb shell am start -n com.example.myapplication/.MainActivity
```

### 操作流程

1. **打开应用** → 授予权限
2. **查看相机界面** → 右下角有浮动按钮 🍔
3. **点击浮动按钮** → 抽屉从右侧滑出
4. **使用快捷功能** → 点击"拍照"等按钮
5. **测试开发工具** → 点击"测试麦克风"等
6. **关闭抽屉** → 点击背景或关闭按钮 ✕

## 📝 开发说明

### 添加新按钮

**在快捷功能区添加按钮**：

编辑 `DebugDrawer.kt` 的 `UserFunctionSection`：

```kotlin
Row(
    modifier = Modifier.fillMaxWidth(),
    horizontalArrangement = Arrangement.spacedBy(8.dp)
) {
    Button(
        onClick = onYourNewFunction,
        modifier = Modifier.weight(1f)
    ) {
        Text("新功能")
    }
    // ... 其他按钮
}
```

**在开发者工具区添加按钮**：

编辑 `DebugDrawer.kt` 的 `DeveloperTestSection`：

```kotlin
OutlinedButton(
    onClick = onYourTestFunction,
    modifier = Modifier.weight(1f)
) {
    Text("测试功能", fontSize = 12.sp)
}
```

### 修改抽屉样式

**调整抽屉宽度**：

```kotlin
Surface(
    modifier = Modifier
        .fillMaxHeight()
        .width(400.dp) // 修改这里
        .align(Alignment.CenterEnd)
    // ...
)
```

**修改背景透明度**：

```kotlin
.background(Color.Black.copy(alpha = 0.7f)) // 修改这里
```
