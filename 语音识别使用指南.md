# 🎤 语音识别使用指南

## 🆕 新功能说明

你的应用现在增加了以下功能来解决识别问题：

### 1. 语言切换功能
- **当前语言显示**：调试面板顶部显示"当前语言: 中文"或"当前语言: 英文"
- **切换按钮**：点击"切换语言"按钮在中文和英文之间切换
- **自动提示**：切换后会语音播报"已切换到XX识别"

### 2. 识别测试功能
- **测试识别按钮**：点击"测试识别"进行一次性识别测试
- **详细反馈**：显示识别过程的每一步（准备→检测到说话→处理中→结果）
- **成功/失败提示**：清楚地显示识别是否成功以及具体结果

### 3. 增强的识别配置
- **更长的超时时间**：识别等待时间从默认增加到 5 秒
- **更多识别结果**：从 1 个增加到 5 个，提高识别成功率
- **更详细的日志**：所有识别过程都会记录在日志中

## 📱 使用步骤

### 步骤 1: 测试麦克风（已完成）

✅ 如果"测试麦克风"显示成功，说明硬件工作正常，可以继续。

### 步骤 2: 选择语言

**重要：根据你要说的语言选择！**

- 如果要说**中文**，确保显示"当前语言: 中文"
- 如果要说**英文**，点击"切换语言"切换到英文

**常见错误**：
- ❌ 设置中文却说英文 → 无法识别
- ❌ 设置英文却说中文 → 无法识别
- ✅ 语言匹配 → 识别成功

### 步骤 3: 测试识别

1. 点击 **"测试识别"** 按钮
2. 看到 "开始识别测试，请说话..."
3. **清晰地说一句话**（例如：）
   - 中文："拍照"、"暂停录像"、"查询电池"
   - 英文："take photo"、"pause recording"、"check battery"
4. 等待识别结果

**成功示例**：
```
✅ 识别成功: 拍照
```

**失败示例**：
```
❌ 识别失败，请检查网络和语言设置
```

### 步骤 4: 如果识别失败

#### 方法 1: 切换语言重试

可能你选错了语言：
1. 点击"切换语言"
2. 再次点击"测试识别"
3. 用新语言说话

#### 方法 2: 检查网络

Google 语音识别需要网络连接：
```bash
# 检查模拟器网络
adb shell ping -c 3 8.8.8.8

# 如果失败，检查模拟器设置 > 网络
```

#### 方法 3: 查看详细日志

```bash
# 实时查看识别过程
adb logcat -s VoiceRecognitionService | grep -E "测试|识别|结果|错误"

# 应该看到：
# [测试] 准备接收语音
# [测试] 检测到说话
# [测试] 识别结果: 你说的话
```

## 🔍 诊断错误类型

### 错误 1: "未识别到语音内容"

**原因**：Google 语音服务接收到声音但无法识别

**解决方案**：
1. 切换语言（最常见原因）
2. 说得更清晰一些
3. 检查网络连接
4. 尝试简单的词汇（如"你好"或"hello"）

### 错误 2: "语音超时（未检测到声音）"

**原因**：识别器没有接收到声音

**解决方案**：
1. 确认"测试麦克风"成功
2. 说话音量提高
3. 距离麦克风近一些

### 错误 3: "网络错误"或"服务器错误"

**原因**：无法连接 Google 服务器

**解决方案**：
1. 检查模拟器网络连接
2. 检查防火墙设置
3. 尝试重启模拟器
4. 考虑使用真实设备

### 错误 4: "客户端错误"

**原因**：识别器忙碌或配置问题

**解决方案**：
1. 等待几秒后重试
2. 重启应用
3. 如果持续出现，使用真实设备

## 💡 使用技巧

### 技巧 1: 先用简单词汇测试

**中文**：先说"你好"、"谢谢"这样的常用词
**英文**：先说"hello"、"yes"这样的简单词

如果简单词汇能识别，说明系统正常，可以继续测试应用命令。

### 技巧 2: 清晰发音

- 不要太快
- 不要太小声
- 避免环境噪音
- 对着麦克风说话

### 技巧 3: 使用持续监听模式

测试成功后，应用会自动进入持续监听模式：
- 自动监听你的语音
- 识别到命令会自动执行
- 无需每次点击按钮

### 技巧 4: 观察调试面板

调试面板会实时显示：
- 🎤 监听状态
- 音量等级（说话时应该有变化）
- 识别的文本
- 错误信息

## 🚨 常见问题

### Q1: 麦克风测试成功，但识别总是失败？

**A**: 最可能的原因是语言设置错误。
- 确认"当前语言"和你说的语言一致
- 尝试两种语言都测试一下

### Q2: 部分结果显示了文字，但最终失败？

**A**: 这说明识别过程在工作，但最终无法确定结果。
- 说得再清晰一些
- 说完后稍等 1-2 秒
- 尝试更简单的词汇

### Q3: 中文和英文都无法识别？

**A**: 可能是网络问题或 Google 服务问题。
- 检查网络连接
- 使用真实设备测试（推荐）
- 考虑使用离线识别方案

### Q4: 在真实设备上会更好吗？

**A**: 是的！真实设备通常效果更好：
- 不需要配置虚拟麦克风
- Google 服务更稳定
- 网络连接更可靠
- 真实硬件麦克风
